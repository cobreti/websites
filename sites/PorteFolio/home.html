NyxTraceViewer

This is a personnal project I started to help debugging with traces when building an application.

It allows reception and display of traces from many modules. Display using word and line highlighting in many colors.

Works on Linux, Windows and OS X

Nyx library

Home made cross-platform library (Windows, OS X and Linux) used to help for my personal C++ projects.

Used by NyxTraceViewer project.

Interview Manager

This is a project made in a Client-Server course for my master degree.

It consisted of a server made in Java connected to an sql database and a graphical user interface made using C# and WPF.

For simplicity reasons, the connection part to the server has been replaced by stubs to allow the client to work by itself without any installation required.
